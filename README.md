# notes-grade-hth

## 项目简介

`notes-grade-hth` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容不针对任何特定域名或网站，仅供个人学习、记录和分享使用。本仓库旨在以静态 HTML 形式保存一些笔记、资料或工具页面，方便通过 GitHub Pages 等方式直接访问。

## 目录说明

```
notes-grade-hth/
├── docs/           # 存放所有 HTML 页面文件
├── assets/         # 页面依赖的静态资源（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `docs/` 目录为默认的页面发布目录，每个独立的 HTML 文件对应一个归档页面。
- `assets/` 目录用于存放页面所需的公共资源，按需添加。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面可以包含笔记、工具、参考资料或其他内容。页面之间没有强关联，可单独访问。所有页面均以静态 HTML 编写，不依赖后端服务。

如需添加新页面，请将 HTML 文件放入 `docs/` 目录，并确保资源路径正确。建议每个页面自包含样式与脚本，或统一引用 `assets/` 中的资源。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎通过 Issues 提交建议或发现问题。
- 如需贡献内容，请 Fork 后提交 Pull Request，确保新增页面不包含任何商业或诱导性内容。
- 页面内容应保持简洁、清晰，避免引入外部跟踪或广告脚本。

## 许可

本仓库内容仅供学习与存档使用，无特定开源协议限制。请合理使用。
